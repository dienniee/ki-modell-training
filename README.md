# ki_projekt
Uni-Projektzum Training eines KI-Modells

In diesem Projekt geht es um das Training eines einfachen KI-Modells sowie um die Viesualisierung der Vorhersageergebnisse. Das Modell wird außerdem auf mögliche Bias überprüft. und im Falle eines festgestellten Bias wird eine ausgewählte Methode zur reduzierung angewendet. 

Technologien:
 - Python
 - Google Colab
 - verwendete Methode zut Bias-reduzierung: Oversampling.
